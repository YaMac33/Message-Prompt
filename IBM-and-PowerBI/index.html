「IBM Data Analyst Professional Certificate」→「Power BI（PL-300）」への流れ
未経験からテレワークや高収入を目指すステップ

目次

🎯 目的

段階的にスキルを習得し、データアナリストまたはBIエンジニアとしてリモートで働けるスキルと資格を取得します。

目標資格：

  • ✅ IBM Data Analyst Professional Certificate（入門編）
  • ✅ Microsoft Power BI Data Analyst Associate（PL-300）（実務対応）

🗓️ ロードマップ：3～6ヶ月プラン

期間 目標内容 学習時間目安（週）
1ヶ月目 IBM Data Analyst開始
CourseraでSQL・Excel・Python・可視化の基礎を学習		 8〜10時間
2ヶ月目 プロジェクト・GitHub開始
IBMの最終プロジェクトでポートフォリオ作成開始		 8〜12時間
3ヶ月目 Power BI入門学習開始
UdemyやMicrosoft LearnでPower BIの基本操作		 10時間
4ヶ月目 DAX・Power Query強化
複雑な集計、フィルタ、変換を学習しダッシュボード制作		 10〜12時間
5ヶ月目 PL-300試験対策
模擬問題＋過去問、公式ラーニングパス完走		 12時間
6ヶ月目 PL-300試験受験 & ポートフォリオ更新
Power BI実績まとめ（GitHub, Tableau Publicなど）		 6〜8時間

📚 使う教材（具体的リンク付き）

✅ IBM Data Analyst（1〜2ヶ月目）

Coursera: IBM Data Analyst Professional Certificate

  • 学べる内容：SQL（DB2）、Excel、Python（Pandas）、可視化（Matplotlib）
  • 実務未経験でも取り組みやすい
  • ポートフォリオ作成にもつながる

✅ Power BI + PL-300（3〜6ヶ月目）

🧑‍🏫 学習教材

  • Microsoft Learn: PL-300ラーニングパス（無料・公式）
  • Udemy: Power BI完全マスターコース（日本語）
  • Udemy: PL-300 Exam Prep（英語）（模試に最適）

📘 PL-300 試験概要

  • 試験名：PL-300: Microsoft Power BI Data Analyst
  • 費用：約 $165（日本円で20,000円前後）
  • 試験方式：オンライン受験可能（英語・日本語）
  • 合格ライン：700/1000点

💡 アドバイス：合格と実務へのつなげ方

実務的な視点を身につけることが、合格とキャリアアップの鍵となります。

ポイント アドバイス
✅ ポートフォリオ IBMとPower BIで作った分析・可視化をGitHubやTableau Publicにまとめる
✅ 模擬試験で慣れる PL-300はシナリオ問題が多いため、実務的な視点が問われる
✅ 実務体験を積む ココナラ・クラウドワークス・Kaggleで軽い案件や課題に挑戦すると◎
✅ LinkedIn活用 英語環境や外資系も狙うなら、LinkedInでプロフィール・成果物を公開

✅ 終了後のキャリアステップ（PL-300取得後）

ステップ 進め方
💼 転職活動開始 リモートOKな「データアナリスト」職に応募開始
🧠 次の学習 DAX上級編・Azure系資格（DP-900など）・ETLツール学習
💰 単価アップ スキルが実績で証明できれば、年収500万〜800万円帯も視野に

🔚 最後に

このロードマップは、「未経験 → 資格取得 → テレワーク可能なBI職」への実践的な道筋です。

無理のないペースで進めて、資格＋スキル＋成果物の3点セットを整えれば、確実にキャリアの選択肢が広がります。
